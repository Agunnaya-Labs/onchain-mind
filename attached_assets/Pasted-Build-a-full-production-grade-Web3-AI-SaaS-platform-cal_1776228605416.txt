Build a full production-grade Web3 AI SaaS platform called “OnchainMind × AGL”.

This is a cloud-native system that allows developers to create AI-powered NPCs with persistent memory and tokenized compute using the AGL token on Base.

----------------------------------

🎯 CORE PRODUCT

A developer platform (like OpenAI) where users can:

- Create organizations and projects
- Generate API keys
- Create and manage NPCs
- Interact with NPCs via API or dashboard
- Track usage and billing in AGL tokens
- View memory logs and analytics

----------------------------------

🧠 CORE FEATURES

1. NPC AI ENGINE
- Endpoint: POST /v1/npc/chat
- Input: npcId, message, wallet
- Output: AI response
- Must simulate memory recall (“I remember…”)
- Integrate OpenAI or mock AI responses

2. MEMORY SYSTEM (ZILLIZ-STYLE)
- Store:
  npcId, projectId, wallet, text, embedding
- Support semantic search
- Multi-tenant filtering:
  projectId + npcId

3. BILLING ENGINE (AGL TOKEN)
- Charge per action:
  chat: 2 AGL
  memory write: 1 AGL
  search: 1 AGL
- Track usage per project
- Display balance in dashboard

4. AUTH SYSTEM
- Wallet login (SIWE-ready)
- API key authentication
- Multi-tenant isolation

5. INDEXER SYSTEM (SIMULATED)
- Simulate blockchain events updating NPC memory
